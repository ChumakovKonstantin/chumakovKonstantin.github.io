⚡ AICHI BOOST 🚀

🎮 Что это за зверь?

AICHI BOOST — это мощное десктопное приложение для оптимизации Windows, которое я разработал на Python + Eel.

Программа позволяет в пару кликов:

  • ✅ Отключить телеметрию Windows
  • ✅ Очистить временные файлы и кэш
  • ✅ Настроить схему питания для максимальной производительности
  • ✅ Отключить ненужные службы (Cortana, Game Bar и др.)
  • ✅ Заблокировать доступ приложений к личным данным
  • ✅ Создать точку восстановления системы

🛠️ Технологии

Python Eel JavaScript HTML/CSS Particles.js Windows Registry API PowerShell win32gui

Интерфейс на HTML/CSS/JS, бэкенд на Python с прямым доступом к системному реестру Windows и службам через win32api.
